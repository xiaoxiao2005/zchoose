# 数据表/模型说明

当前使用 SQLite（sql.js），库文件：`data/app.db`。

## 表结构

### users 用户

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| phone | TEXT UNIQUE | 手机号（可选） |
| email | TEXT UNIQUE | 邮箱（可选） |
| password_hash | TEXT | 密码哈希（bcrypt），必填 |
| wechat_openid | TEXT UNIQUE | 微信 openid（预留） |
| qq_openid | TEXT UNIQUE | QQ openid（预留） |
| created_at | TEXT | 创建时间 |

### verification_codes 注册验证码

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| target | TEXT | 邮箱或手机号 |
| type | TEXT | email / phone |
| code | TEXT | 6 位数字验证码 |
| expires_at | TEXT | 过期时间（5 分钟有效） |
| created_at | TEXT | 创建时间 |

用于 `POST /api/users/send-code` 写入、`POST /api/users/register` 校验后删除。当前未接短信/邮件时，开发环境响应中返回 `code` 供测试。

### body_profiles 用户体型（与试衣生成关联）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER UNIQUE | 用户 ID |
| gender | TEXT | 性别 |
| height_cm | INTEGER | 身高(cm) |
| weight_kg | INTEGER | 体重(kg) |
| body_type | TEXT | 体型选项：slim/standard/slight/heavy |
| prompt_snippet | TEXT | 体型对应提示词片段 |
| extra_prompt | TEXT | 其他特征提示词（用户自定义，如肩宽、腿长等） |
| updated_at | TEXT | 更新时间 |

### outfits 衣库（一套搭配）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| name | TEXT | 名称 |
| image_url | TEXT | 展示图 URL |
| style_tags | TEXT | 风格标签，逗号分隔 |
| need_points | INTEGER | 解锁所需积分，0 表示免费 |
| created_at | TEXT | 创建时间 |

### user_points 用户积分

| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | INTEGER PK | 用户 ID |
| points | INTEGER | 当前积分 |
| updated_at | TEXT | 更新时间 |

### user_unlocks 用户已解锁搭配

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER | 用户 ID |
| outfit_id | INTEGER | 搭配 ID |
| unlocked_at | TEXT | 解锁时间 |

### tryon_results 试衣生成结果

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键（用于后期美化） |
| user_id | INTEGER | 用户 ID |
| outfit_id | INTEGER | 搭配 ID |
| photo_url | TEXT | 用户上传照片 URL |
| front_url, side_url, back_url | TEXT | 三视图 URL |
| created_at | TEXT | 创建时间 |

### outfit_records 穿搭记录（用户画像）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER | 用户 ID |
| outfit_id | INTEGER | 搭配 ID |
| occasion | TEXT | 场合 |
| weather_temp | REAL | 气温 |
| weather_desc | TEXT | 天气描述 |
| created_at | TEXT | 创建时间 |

### user_submissions 用户投稿搭配

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER | 用户 ID |
| image_url | TEXT | 投稿图 URL |
| description | TEXT | 描述 |
| status | TEXT | pending / accepted / rejected |
| created_at | TEXT | 投稿时间 |
| reviewed_at | TEXT | 审核时间 |

### support_messages 客服对话与转人工/留言

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER | 用户 ID |
| role | TEXT | user / assistant / system |
| content | TEXT | 文本内容 |
| image_url | TEXT | 图片 URL（可选） |
| is_transfer_human | INTEGER | 是否转人工/留言（0/1） |
| created_at | TEXT | 创建时间 |

### douyin_claims 抖音分享核销

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| user_id | INTEGER | 用户 ID |
| link | TEXT | 抖音链接（可选） |
| image_url | TEXT | 截图 URL（可选） |
| status | TEXT | pending / approved / rejected |
| outfit_id | INTEGER | 通过后解锁的搭配 ID |
| created_at | TEXT | 提交时间 |
| reviewed_at | TEXT | 核验时间 |

### password_reset_tokens 忘记密码重置

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| email | TEXT | 邮箱 |
| token | TEXT UNIQUE | 重置 token |
| expires_at | TEXT | 过期时间（1 小时） |
| created_at | TEXT | 创建时间 |
