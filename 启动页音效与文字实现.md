# 启动页（Splash）音效与文字实现

本文档单独整理启动页的**音效**与**品牌文字**相关代码，便于复用或修改。

---

## 一、逻辑与时序

- **音效**：使用 `public/sounds/startup.mp3`，进入页面即预加载，约 0ms 开始播放。
- **文字**：品牌名「Zchoose」在**音效播放约 1 秒后**开始亮起（brighten），再停留、淡出，最后跳转首页。

| 阶段       | 含义           | 相对时间（ms） |
|------------|----------------|----------------|
| `black`    | 黑屏，仅播音乐 | 0 ~ 1000       |
| `brighten` | 文字渐显       | 1000 ~ 1500    |
| `hold`     | 文字保持可见   | 1500 ~ 2700    |
| `fadeout`  | 文字淡出       | 2700 ~ 3100    |
| 跳转 `/home` | -            | 3100           |

## 二、TSX 组件代码（`frontend/src/pages/Splash.tsx`）

```tsx
import { useEffect, useRef, useState } from 'react';
import { useNavigate } from 'react-router-dom';
import './Splash.css';

type Phase = 'black' | 'brighten' | 'hold' | 'fadeout';

const TIMING = {
  soundFirst: 1000,   // 音乐先播放 1 秒，再出文字
  brighten: 500,
  hold: 1200,
  fadeout: 400,
};

/** 启动音效：public/sounds/startup.mp3，先播 1 秒后出字 */
const STARTUP_SOUND_URL = '/sounds/startup.mp3';

export default function Splash() {
  const [phase, setPhase] = useState<Phase>('black');
  const navigate = useNavigate();
  const soundPlayed = useRef(false);
  const audioRef = useRef<HTMLAudioElement | null>(null);

  // 进入页面即预加载音效，避免播放时加载导致与文字不同步
  useEffect(() => {
    try {
      const audio = new Audio(STARTUP_SOUND_URL);
      audio.volume = 0.7;
      audio.preload = 'auto';
      audio.load();
      audioRef.current = audio;
      return () => {
        audioRef.current = null;
      };
    } catch {
      audioRef.current = null;
    }
  }, []);

  useEffect(() => {
    // 音乐先播放 1 秒，再出现文字
    const t0 = setTimeout(() => {
      if (!soundPlayed.current && audioRef.current) {
        soundPlayed.current = true;
        audioRef.current.currentTime = 0;
        audioRef.current.play().catch(() => {});
      }
    }, 0);
    const t1 = setTimeout(() => setPhase('brighten'), TIMING.soundFirst);
    const t2 = setTimeout(() => setPhase('hold'), TIMING.soundFirst + TIMING.brighten);
    const t3 = setTimeout(
      () => setPhase('fadeout'),
      TIMING.soundFirst + TIMING.brighten + TIMING.hold
    );
    const t4 = setTimeout(
      () => navigate('/home', { replace: true }),
      TIMING.soundFirst + TIMING.brighten + TIMING.hold + TIMING.fadeout
    );
    return () => {
      clearTimeout(t0);
      clearTimeout(t1);
      clearTimeout(t2);
      clearTimeout(t3);
      clearTimeout(t4);
    };
  }, [navigate]);

  return (
    <div
      className={`splash splash--${phase}`}
      role="presentation"
      aria-label="Zchoose 启动中"
    >
      <div className="splash__bg" />
      {/* 【界面美化-可替换】品牌名、发光样式 */}
      <span className="splash__logo">Zchoose</span>
    </div>
  );
}
```

## 三、样式代码（`frontend/src/pages/Splash.css`）

/* 启动闪屏：黑屏 → 仅 Zchoose 发光出现 → 停留 → 淡出，背景保持纯黑不刺眼 */

## 四、音效文件与可调参数

- **音效路径**：`frontend/public/sounds/startup.mp3`  
  - 若没有 `public/sounds` 目录，需新建并把 `startup.mp3` 放进去；开发环境访问路径为 `/sounds/startup.mp3`。
- **音量**：代码里 `audio.volume = 0.7`，可按需改为 0～1。
- **时序**：在 `TIMING` 中调整：
  - `soundFirst`：音效先播多久再出字（当前 1000ms）
  - `brighten`：文字渐显时长
  - `hold`：文字停留时长
  - `fadeout`：淡出时长，结束后即跳转首页。

以上即为启动页音效与文字部分的完整实现代码与说明。
