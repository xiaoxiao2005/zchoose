# 前端界面美化批注说明

这里的「界面美化」指**页面上按钮、文字、图片、配色等视觉优化**（不是 AI 修图）。  
在代码里已用统一标记标出**你可以直接替换**的位置，方便改成自己的风格。

---

## 批注标记

在 HTML、Vue、React 等文件中，用以下格式标注：

- **`【界面美化-可替换】`** + 简短说明（如：按钮文案、标题、图片路径、主题色）

替换时只需改对应那一行或那一块内容即可。

---

## 可替换内容类型

| 类型 | 说明 | 示例位置 |
|------|------|----------|
| **按钮文案** | 按钮上显示的文字 | 「立即试穿」「提交」「刷新」等 |
| **标题/副标题** | 页面标题、区块标题、说明文字 | 首页主标题、卡片标题、提示语 |
| **图片** | 图标、背景图、占位图、Logo | `src="..."`、背景 `url(...)` |
| **主题色/配色** | 主色、背景色、按钮色、文字色 | CSS 变量或 class |
| **Logo / 品牌名** | 顶部导航品牌、底部版权 | 站点名称、slogan |

---

## 项目里已标出的文件

- `frontend/demo/index.html` — 示例页，含标题、按钮、卡片、配色等，每处可替换都有批注。

后续你用 Vue/React 做正式前端时，在对应组件里按同样方式加 `【界面美化-可替换】` 注释即可，方便以后统一换文案和视觉。
